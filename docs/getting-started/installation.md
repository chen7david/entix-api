# Installation Guide

This guide covers the different ways to set up the Entix API development environment.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v18 or later)
- **npm** (v9 or later)
- **Docker** and **Docker Compose** (for containerized development)
- **Git** (for version control)

## Quick Start with Docker (Recommended)

The easiest way to get started is using Docker, which provides a consistent development environment with all dependencies pre-configured.
